# Saga — Azulejo "Caravel" brand brief (executable)

> **For the Claude session executing this:** this file is a runnable spec, not background reading.
> Work through the **Build steps** in order. The visual target is the image at
> `docs/brand/reference-caravel-panel.jpg` — **open and look at it with the Read tool before drawing anything.**
> Before producing any SVG, load the `frontend-design` skill for aesthetic direction.

---

## 1. The goal

Give the Saga desktop app (Tauri v2, vanilla TS + Vite) a Portuguese-azulejo identity built on the
*caravel* branding it already carries. Two visual artifacts, one shared visual language:

1. **Hero panel** — a full azulejo *painel* (the framed tile scene) for the splash / About screen.
   Faithful to the reference, detailed, beautiful at large sizes.
2. **App icon mark** — a *simplified* descendant of the panel that still reads at 16–32 px.
   This becomes the Windows/macOS/Linux app icon.

Plus a **theme** for the UI derived from the same palette (light "azulejo" + dark "navy" variants).

The hero and the icon must look like the same family — same blue, same ochre accent, same cross — but
the icon is a deliberate reduction, **not** the panel shrunk down (detail dies below ~64 px).

---

## 2. The reference, described

Open `docs/brand/reference-caravel-panel.jpg`. It is a classic blue-monochrome azulejo panel:

- **Subject:** a Portuguese *nau / caravela* under full sail on stylised ocean swells, cloudy sky behind.
  The square sails carry the **Cruz de Cristo** (Order of Christ cross — splayed arms, hollow centre).
  Pennants fly from the mastheads.
- **Palette:** cobalt-blue monochrome painting on a warm off-white glaze. Shading runs from near-navy
  outlines through mid-blue washes to pale sky-blue. **Not** pure black/white — it is hand-painted blue.
- **Border (cercadura):** a thick **guilloche / rope-braid** running the full frame — interlocking blue
  loops with **ochre-orange** beads woven through them. This ochre is the *only* warm colour; it is the
  signature accent.
- **Corners:** square tiles with a stylised blue **cross-floral ornament** (fleur/cross motif) on white.
- **Inner frame:** a thin delicate blue filigree/scroll line between border and scene.
- **Tile grid:** faint seams divide the panel into individual square tiles — keep this; it reads as
  "real azulejo," not a sticker.

## 3. Palette (extract / tune against the actual image)

These are the design tokens. Sample the reference to refine, but stay in this family:

| Role                  | Hex        | Use                                              |
|-----------------------|------------|--------------------------------------------------|
| Cobalt (line/navy)    | `#1c3f73`  | Outlines, frame, darkest shading, primary text   |
| Mid blue              | `#2f6ea5`  | Hull/wave washes, UI accent                      |
| Light cobalt          | `#6fa8d6`  | Sky wash, dark-theme accent                      |
| Pale sky              | `#bcd6e8`  | Lightest wash, hover fills                       |
| Tile white (glaze)    | `#f4f1e9`  | Warm off-white ground (never pure #fff)          |
| Ochre (rope bead)     | `#d98e3c`  | THE accent — border beads, highlights, CTA       |
| Deep ochre            | `#c8782a`  | Ochre shadow / pressed state                     |

## 5. Build steps (do these in order)

### Step 0 — Look & set direction
- Read `docs/brand/reference-caravel-panel.jpg`.
- Load the `frontend-design` skill.

### Step 1 — Hero panel SVG → `assets/brand/caravel-panel.svg`
- Hand-author SVG (this is vector illustration, not a generator). Build it in layers:
  ground → tile-grid seams → outer rope border (with ochre beads) → corner cross-florals →
  inner filigree frame → sky/clouds → sea swells → ship hull → masts/rigging → sails with
  **Cruz de Cristo** → pennants.
- Aspect ~2:3 (portrait), like the reference. Use only the palette above.
- Render it in an **Artifact** to preview and iterate with Gabriel before moving on.

### Step 2 — Icon mark SVG → `assets/brand/caravel-mark.svg`
- A square (1:1) reduction that survives at 32 px. Pick ONE strong idea, e.g.:
  - a single caravel silhouette with **one** Cruz-de-Cristo sail, on three wave-curls, inside a
    rounded tile with a *thin* rope border; **or**
  - just the Cruz de Cristo over stylised waves if the ship is too busy small.
- Bold shapes, max 2–3 blue values + ochre. Test legibility by viewing it small in the Artifact.
- Keep the background **transparent** (the OS rounds/frames icons itself).

### Step 3 — Rasterise the mark to 1024 PNG
Pick whichever tool is present (transparent background, 1024×1024):
```bash
# option A — resvg (crisp, recommended)
npx @resvg/resvg-js-cli assets/brand/caravel-mark.svg assets/brand/caravel-mark-1024.png --width 1024 --height 1024
# option B — sharp (already a node project)
node -e "require('sharp')('assets/brand/caravel-mark.svg',{density:300}).resize(1024,1024).png().toFile('assets/brand/caravel-mark-1024.png')"
# option C — rsvg-convert (if installed)
rsvg-convert -w 1024 -h 1024 assets/brand/caravel-mark.svg -o assets/brand/caravel-mark-1024.png
```

### Step 4 — Generate the platform icon set
Tauri builds every format (.ico, .icns, all PNGs) from the one PNG:
```bash
npx @tauri-apps/cli icon assets/brand/caravel-mark-1024.png
```
This overwrites `src-tauri/icons/*`. The paths already referenced in `src-tauri/tauri.conf.json`
(`32x32.png`, `128x128.png`, `128x128@2x.png`, `icon.icns`, `icon.ico`) stay valid — no config edit needed.

### Step 5 — Theme `src/style.css`
The current `:root` is a dark near-black theme. Add azulejo theming **without breaking the token names**
already used across the app (`--bg`, `--bg-2`, `--bg-3`, `--border`, `--text`, `--muted`, `--accent`,
`--local`, `--claude`, `--danger`, `--radius`).

Default to the **light azulejo** theme (tiles are light) and provide the **dark navy** under
`prefers-color-scheme: dark` (and/or a `[data-theme]` toggle if the app has one):

```css
:root {                  /* Light — azulejo glaze */
  --bg:    #f4f1e9;
  --bg-2:  #eaf0f4;
  --bg-3:  #dde8f0;
  --border:#b9cdde;
  --text:  #1c3f73;      /* cobalt ink */
  --muted: #5a7596;
  --accent:#2f6ea5;      /* mid cobalt */
  --local: #4f8a6d;      /* polychrome tile green (semantic: local) */
  --claude:#d98e3c;      /* ochre rope bead (semantic: claude) */
  --danger:#c0413b;
  --radius:12px;
}
@media (prefers-color-scheme: dark) {
  :root {                /* Dark — cobalt night */
    --bg:    #0e1d33;
    --bg-2:  #13294a;
    --bg-3:  #1a3357;
    --border:#29456f;
    --text:  #eaf1f7;    /* tile white */
    --muted: #8fa6c2;
    --accent:#6fa8d6;    /* light cobalt */
    --local: #5fae87;
    --claude:#e0a45a;    /* ochre */
    --danger:#f08a8a;
    --radius:12px;
  }
}
```
Preserve `--radius` and any tokens not listed here. After editing, scan `src/style.css` and `src/*.ts`
for hard-coded colours that should reference these vars instead.

### Step 6 — Build & verify
```bash
npm run tauri dev      # confirm UI theme + window icon
npm run tauri build    # confirm installer/app icons on this platform
```
Use the `run` or `verify` skill to launch and eyeball it. Check the icon at the taskbar/dock size —
if the mark is mushy small, return to Step 2 and simplify further.

## 6. Acceptance checklist
- [ ] Hero panel reads as a real azulejo *painel* (rope border + ochre beads + corner florals + tile seams + Cruz-de-Cristo sails).
- [ ] Icon mark is legible at 32 px and 16 px; clearly the same family as the hero.
- [ ] Palette holds the line: warm-white glaze, cobalt range, single ochre accent — no stray colours.
- [ ] `tauri icon` regenerated all `src-tauri/icons/*`; app/window icon shows the new mark.
- [ ] Light + dark themes both legible; existing token names untouched; no hard-coded colours left.
- [ ] `npm run tauri build` succeeds.

## 7. Notes
- Keep `caravel-panel.svg` / `caravel-mark.svg` as the editable masters — regenerate rasters/icons from them.
- If a richer painterly hero is wanted later, the SVG can be taken to Recraft (real SVG output) or an
  image model for a textured version — but the SVG masters stay the source of truth.
